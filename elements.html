MES PROJETS

Projets réalisés en entreprise

Projet 1 : Réseau social d’entreprise

En équipe, nous avons conçu une application web en utilisant Eclipse et Java pour créer un réseau social destiné aux entreprises. Cette plateforme vise à faciliter la communication, le partage d’informations et la collaboration entre les entreprises de manière sécurisée.

Projet 2 : Recherche de vulnérabilités

Dans le cadre de mon projet professionnel, j’ai élaboré un programme axé sur la cybersécurité. Ce programme est capable de rechercher les vulnérabilités d’un site web par une recherche par mots-clés.

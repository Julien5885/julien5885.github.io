MES PROJETS

Projets réalisés à l'école

Projet 1 : JumpX

JumpX est la toute première application mobile que nous avons créée en équipe durant la première année de notre BTS SIO option SLAM. Ce projet nous a permis de mettre en pratique toutes les compétences demandées dans le cadre du Bloc 1 du BTS, à savoir :

  • Analyse des besoins et de la demande du client.
  • Utilisation du logiciel GANTT pour planifier et organiser le projet.
  • Réalisation du Modèle Conceptuel de Données (MCD) pour structurer les bases de données.
  • Création des maquettes pour représenter visuellement l’application avant son développement.

JumpX est une plateforme conçue avec Eclipse et Java, destinée à être un réseau social pour entreprises, facilitant la communication et la collaboration de manière sécurisée.

Projet 2 : Projet de deuxième année - RFTG

RFTG (Rental For The Gérant) est une application de gestion de location de DVD développée pour répondre aux besoins d’un centre de vacances. Ce projet, réalisé en deuxième année, est divisé en deux parties distinctes :

  • Mario : une application web destinée à la gestion du stock des DVD, utilisée par le gérant et ses employés.
  • Luigi : une application mobile permettant aux clients du centre de vacances de louer des DVD facilement.

Ce projet nous a permis de travailler sur plusieurs technologies et outils avancés :

  • Back-end en Spring Boot pour assurer une API robuste et performante.
  • Front-end en Laravel pour une interface ergonomique et moderne.
  • Utilisation de VS Code et XAMPP pour le développement.
  • Réalisation des maquettes et du MCD pour définir clairement l’architecture du projet.

Ce projet nous a permis de consolider nos compétences en développement web et mobile, tout en mettant en place une solution complète et fonctionnelle adaptée aux besoins du client.

Projet 1 : Application de gestion de bibliothèque

Durant mon expérience en entreprise, j’ai eu l’opportunité de travailler sur une application de gestion de bibliothèque développée en Java avec une base de données MySQL.
